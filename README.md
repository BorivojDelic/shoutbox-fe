# Shoutbox Frontend

## Overview

This is the frontend application for the Shoutbox built with React.

## Getting Started

### Prerequisites

*   Node.js (v18+)

### Install
```bash
  npm install
```

### Run

```bash
  npm start
```

This starts the development server.

## Technical Overview

*   React + CSS
*   `socket.io-client` (Real time communication)
*   `axios` (API requests)
*   Custom Hooks & Context for socket management
